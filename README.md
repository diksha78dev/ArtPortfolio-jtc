## 🧠 About The Project

**Jiya The Coolartist** is a personal art portfolio website that serves as a digital gallery and learning hub for aspiring artists. This project showcases various art styles, provides educational resources, and tells the inspiring journey of an artist who started at 16.

### What Makes This Portfolio Special?

- 🎨 **Diverse Art Styles**: Oil Pastel, Watercolor, Pencil Shading, Mandala, and Doodle art
- 🧰 **Practical Resources**: Detailed material lists with budget-friendly alternatives
- 📹 **Multimedia Integration**: Embedded tutorials and social media links
- 📖 **Personal Storytelling**: Authentic journey from beginner to established artist
- 🌐 **Fully Responsive**: Optimized for viewing on all device sizes

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser to view and edit this project:

- Chrome (recommended) v90+
- Firefox v88+
- Safari v14+
- Edge v90+

### Installation & Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/JiyaBatra/ArtPortfolio-jtc.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd ArtPortfolio-jtc
   ```

3. **Open in browser**
   
   Simply open `index.html` in your preferred browser:
   
   ```bash
   # On macOS
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
   ```

4. **Or use a local server (recommended)**
   
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (npx)
   npx serve
   ```
   
   Then visit `http://localhost:8000` in your browser.

### Deployment

This project is automatically deployed via GitHub Pages. Any push to the `main` branch will trigger a deployment.

To deploy your own version:

1. Fork this repository
2. Go to repository Settings → Pages
3. Select branch: `main`
4. Select folder: `/ (root)`
5. Click Save

Your site will be live at: `https://[your-username].github.io/ArtPortfolio-jtc/`
